Why Jestronaut

Everything Jest output should be

⚡

Live dashboard

Watch suites complete in real time with animated spinners and instant result updates — no more scrolling a wall of text.

🔍

Keyboard navigation

Arrow keys, Tab, Enter. Browse suites and individual test results without touching the mouse. Full keybinding help with ?.

💥

Failure drill-down

Open any failing test to see the full error message and stack trace in a scrollable overlay — right inside the dashboard.

🔌

Zero config

One line in jest.config.js and you're done. No plugins, no wrappers, no config files to maintain.

Quick Start

Up in 60 seconds

1. Install

terminal
npm install --save-dev jestronaut

2. Add to jest.config.js

jest.config.js
// jest.config.js
module.exports = {
  reporters: ['jestronaut'],
};

3. Run via the CLI (suppresses raw Jest output)

terminal
npx jestronaut

Or add it as an npm script in package.json:

package.json
{
  "scripts": {
    "test": "jestronaut"
  }
}
